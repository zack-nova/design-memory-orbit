# 设计记忆宪法：Design Memory

本仓库使用 design memory orbit 保存人与 Agent 讨论设计时形成的长期项目语言与架构决策。

## 不可违反规则

- 先读 `docs/design-memory-orbit/INDEX.md`。
- 项目记忆由 `CONTEXT.md`、`CONTEXT-MAP.md`、`docs/adr/` 及 context 目录内的同名文件组成。
- 不得编造领域术语、上下文边界或架构决策；不确定内容必须保留为 open question。
- 缺少项目记忆不是错误；只在术语或决策被确认后 lazy create。
- 任何输出若与既有 ADR 冲突，必须明确指出冲突并等待人类决策。

更多规则和模板见 `docs/design-memory-orbit/INDEX.md`。
